# AutoClicker
A autoclicker which doesn't break!

**This README covers installation, usage, tech stack, licensing, and also a quick preview of 1,000,000 stimulation on neal.fun (check him out) earned all from my autoclicker**
**You need the chrome web browser**

## Installation
Download the latest release, or git clone for cutting edge changes with
```bash
git clone https://github.com/live-by-unix/auto-clicker.git
```
Unzip the changes if you downloaded the release then go to [chrome://extensions](chrome://extensions)
Click on the Developer Mode toggle if not already enabled. NOTE: Most orginizations like schools block Dev Mode. 
Click on 'Load Unpacked' and select the extension. It should appear. 
Optional: Pin the extensions

## Usage
To use, click on the extension icon and enter in however many ms of delay you want. 
Once set click start go to a webpage (The webpage I tested on is neal.fun's stimulation clicker.)
The extension will stop when you exit the webpage and click on the extension, resetting to the defualt 100 ms delay. 
This extension works on any web page element affected by clicking, SO BE CAREFUL!

## Tech Stack
The tech stack is:
* HTML
* CSS
* JS
* JSON
* Chrome Manifest V3
